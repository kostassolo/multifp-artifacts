1. Enable hotkey for switching the addon ON | OFF (Crtl+Shift+Y)
2. Enable hotkey for adding the active tab URL to the whitelist table (Crtl+Shift+U)
3. Enable hotkey for removing the active tab URL from the whitelist table (Crtl+Shift+Z)
Color Temperature (Change Lux) protects your eyes by adding a warm color to all websites (i.e. at night time or when the ambient is dark). Using this add-on, you can avoid excessive light, and also it makes you sleep much better by protecting you from the - blue glow - of your computer's screen. To add a website to the whitelist table, enter the website address in the input field below and click on the (+) sign or press enter. Please note: the address should be a valid URL (i.e. https://www.google.com/), otherwise, it won't be added to the list.
